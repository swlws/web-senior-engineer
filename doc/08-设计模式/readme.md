# 设计模式

## 设计模式分类总览

| 分类                  | 目的            | 常见模式                                   |
| ------------------- | ------------- | -------------------------------------- |
| **创建型（Creational）** | 负责对象的创建       | 单例、工厂、抽象工厂、建造者、原型                      |
| **结构型（Structural）** | 负责对象的组合与结构    | 适配器、装饰器、代理、外观、组合、享元                    |
| **行为型（Behavioral）** | 负责对象间的通信与职责分配 | 观察者、发布订阅、策略、命令、状态、责任链、迭代器、中介者、备忘录、模板方法 |

## 前端开发中最常见的 6 大模式

| 模式     | 实际应用                       |
| ------ | -------------------------- |
| 单例模式   | Vuex / Redux store、全局配置    |
| 工厂模式   | createApp、createVNode、组件工厂 |
| 观察者模式  | Vue2 响应式系统                 |
| 发布订阅模式 | 事件总线（EventBus）             |
| 代理模式   | Vue3 响应式（Proxy）            |
| 策略模式   | 表单验证、多种计算逻辑                |
